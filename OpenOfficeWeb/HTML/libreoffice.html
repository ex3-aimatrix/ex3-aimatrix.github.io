Libre Office

LibreOffice is a successor project to OpenOffice.org, commonly known as OpenOffice. It adds extra features and improved Microsoft Office compatibility, and has regular releases with security updates.

Your OS Is :

Windows 64-Bit
Download LibreOffice For Windows 64-Bit

  1. Go To The Link And Download LibreOffice Installer. (Latest LibreOffice Version 7.1.0)
  2. Open The Installer And Allow LibreOffice To Make Changes To Your Device.
  3. Click Next
  4. Set The Installation To Typical (Default).
  5. Click Install.
  6. Click Finish.

Windows 32-Bit
Download LibreOffice For Windows 32-Bit

  1. Go To The Link And Download LibreOffice Installer. (Latest LibreOffice Version 7.1.0)
  2. Open The Installer And Allow LibreOffice To Make Changes To Your Device.
  3. Click Next
  4. Set The Installation To Typical (Default).
  5. Click Install.
  6. Click Finish.

MacOS
Download LibreOffice For MacOS

  1. Go To The Link And Download LibreOffice Installer. (Latest LibreOffice Version 7.1.0)
  2. Double Click On The DMG File
  3. Drag To Applications Folder When Prompted
  4. When You First Open LibreOffice You May Get An Error As Follows “Libre Office Cannot Be Opened” As The Developer Cannot Be Verified.
  5. Under System Preferences / Security & Privacy / General, There Will Be An Option To Open An App By "Forcing" It To Open. Once Opened, It Will Continue To Be Accessible.
