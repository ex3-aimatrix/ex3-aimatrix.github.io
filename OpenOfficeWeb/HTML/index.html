Open Office Installation Support

Windows
Installation Guide For Windows

  1. Go To The Link And Download OpenOffice Installer. (Latest OpenOffice Version 4.1.8)
  2. Open The Installer And Allow OpenOffice To Make Changes To Your Device.
  3. Click Next And Then Install. (Leave The Directory Set To Default)
  4. Click Next.
  5. Enter Your Username And Click Next.
  6. Set The Installation To Typical (Default).
  7. Click Install.
  8. Click Finish. (Now Follow JRE Installation For Windows)

Mac OS
Installation Guide For Mac OS

  1. Go To The Link And Download OpenOffice Installer. (Latest OpenOffice Version 4.1.8))
  2. Open The Installer And Drag OpenOffice To Application Folder. (Now Follow JRE Installation For Mac)
  3. When You First Open OpenOffice You May Get An Error As Follows “Open Office Cannot Be Opened” As The Developer Cannot Be Verified.
  4. Under System Preferences / Security & Privacy / General, There Will Be An Option To Open An App By "Forcing" It To Open. Once Opened, It Will Continue To Be Accessible.
